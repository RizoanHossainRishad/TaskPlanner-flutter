##  Installation

### 1. Clone the repository
```bash
git clone https://github.com/RizoanHossainRishad/TaskPlanner-flutter.git
```
### 2.**Navigate to the project folder**
   ```bash
   cd TaskPlanner-flutter
   ```
### 3.**Get Flutter dependencies**
   ```bash
   flutter pub get
   ```
### 4.**Run the app**
   ```bash
   flutter run
  ```
